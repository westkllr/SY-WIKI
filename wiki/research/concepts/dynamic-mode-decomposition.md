## 한 줄 요약

실험 데이터의 시간 스냅샷만으로 시스템의 **선형 동역학 모델(A, B)을 SVD 기반으로 자동 추출**하는 방법 — 물리 방정식 없이 데이터에서 "최적 선형 근사"를 얻는다.

---

## 핵심 원리

### DMD (Dynamic Mode Decomposition)

데이터 행렬 X = [x₀, x₁, ..., x_{N-1}] 과 X' = [x₁, x₂, ..., x_N] 이 주어졌을 때:

```
x_{k+1} ≈ A x_k
```

를 만족하는 최적 A를 최소제곱으로 찾는다:

```
A = X' · X†  (X†: 유사역행렬)
```

SVD를 활용해 계산 효율적으로 수행. A의 고유값 = **DMD 모드** → 시스템의 지배적 시공간 패턴.

### DMDc (DMD with control)

입력 u가 있는 시스템:
```
x_{k+1} ≈ A x_k + B u_k
```

확장 데이터 행렬 Ω = [X; U]로 A, B 동시 추정:
```
X' = [A | B] · Ω  →  [A | B] = X' · Ω†
```

→ **제어 시스템에서 쓸 수 있는 상태공간 모델을 데이터로부터 자동 생성**.

### DMDc vs. EDMD (Koopman)

| 항목 | **DMDc** | **EDMD (Koopman)** |
|------|---------|-------------------|
| 작동 공간 | 원래 상태공간 (x) | Lifting된 관측 공간 (φ(x)) |
| 비선형 처리 | 제한적 (선형 근사) | 비선형 시스템 선형화 가능 |
| 차원 변화 | 유지 (n→n) | 증가 (n→N, N>>n) |
| 계산량 | 낮음 | 높음 |
| 물리 내재화 | 없음 | 없음 |

→ DMDc ⊂ EDMD 관계: EDMD에서 φ(x) = x (항등 기저)로 설정하면 DMDc와 같음.

### 배터리 시스템에서의 DMDc (Labib 2026)

배터리 전압 v를 상태로, 전류 i를 입력으로 설정:
```
v_{k+1} ≈ A v_k + B i_k
```
- 30Ah 셀 HPPC 데이터로 A, B 학습
- 데이터 부족 환경에서 Transformer보다 우세
- RC 회로 구조를 명시하지 않고도 동등한 예측 정확도 달성

## 교수님 연구 적용 방안

### 연구 영역 5 — 차량 상태·파라미터 추정

| 시나리오 | DMDc 활용 |
|---------|---------|
| **배터리 ECM 파라미터 동정** | HPPC 실험 데이터로 DMDc 수행 → 2RC 등가회로 파라미터 추출. 기존 EIS(전기화학 임피던스 분광)보다 빠른 현장 동정 가능 |
| **배터리 SOC 추정** | DMDc 선형 모델 위에 칼만 필터 적용 (Koopman-KF) → 물리 구조 없이도 선형 KF 사용 |
| **드라이브트레인 동역학 동정** | 모터-기어-휠 결합계의 A, B를 CAN 데이터로 직접 추정; 기계 공차 변화 반영 |

### 연구 영역 4 — 전동화 파워트레인 최적 제어

- **온라인 DMDc**: 슬라이딩 윈도우로 배터리 모델 실시간 갱신 → 배터리 노화에 따른 MPC 모델 업데이트 자동화
- **에너지 관리**: 주행 패턴 데이터로 DMDc 기반 드라이브 사이클 예측 모델 학습 → 예측 에너지 관리에 활용

### 연구 영역 1 — AI 기반 차량 모델링

FP 모델을 구축하기 어렵거나 파라미터 불확실성이 큰 시스템(예: 연료전지 시스템, 초대용량 커패시터)에서 **구조 가정 없이 데이터만으로 선형 모델** 획득.

## 기존 방법론과 비교

| 항목 | **EKF + 물리 ECM** (기존) | **Dual EKF** (파라미터 추정) | **DMDc** |
|------|------------------------|--------------------------|---------|
| 모델 구조 필요 | 필요 (RC 회로 구조) | 필요 | **불필요** |
| 파라미터 동정 방법 | 사전 실험 + 고정 | 온라인, 상태와 동시 | 오프라인 배치 또는 온라인 |
| 계산 복잡도 | 낮음 | 중간 | 낮음 (SVD) |
| 비선형성 처리 | 선형화 | 선형화 | **선형 근사 (동일 한계)** |
| 노화 적응 | 어려움 | 가능 (느림) | **슬라이딩 윈도우로 빠르게** |
| 외삽 성능 | 물리 기반으로 좋음 | 중간 | **나쁨 (학습 범위 밖)** |
| 이론적 보장 | 칼만 최적성 | 근사 최적 | 없음 (데이터 적합) |

> **핵심 차이 (vs. 기존 EKF)**: 기존 EKF는 "2RC 회로다"라는 물리 구조를 사람이 설계하고 파라미터를 실험으로 맞춤. DMDc는 구조를 전혀 가정하지 않고 데이터가 알아서 선형 근사를 찾음. 장점은 범용성, 단점은 물리적 해석 불가 + 외삽 불신뢰.

### MPC와의 연계

DMDc로 얻은 (A, B)를 선형 MPC의 플랜트 모델로 직접 사용 가능:
```
min Σ ||x - x_ref||²_Q + ||u||²_R
s.t. x_{k+1} = A x_k + B u_k    ← DMDc 모델
```
이것이 **Koopman MPC**와 동일한 구조 (φ(x) = x인 특수 케이스).

## 언제 사용하는가?

### 적합한 상황
- ✅ FP 모델 구조를 모르거나 만들기 어려운 시스템의 초기 탐색적 모델링
- ✅ 빠른 프로토타입: 데이터만 있으면 즉시 선형 모델 획득
- ✅ 배터리 노화처럼 **서서히 변하는 시스템** — 슬라이딩 윈도우 온라인 DMDc
- ✅ 데이터 부족 환경 (신경망보다 소량 데이터에서 강함 — Labib 2026 결과)

### 주의가 필요한 상황
- ⚠️ 강한 비선형 시스템 — EDMD/Koopman으로 업그레이드 필요
- ⚠️ 학습 데이터 범위 밖 외삽 — GPR처럼 불확실성을 알려주지 않음
- ⚠️ 물리적 파라미터 해석이 필요한 경우 — A, B의 원소가 물리 의미를 가지지 않음
- ⚠️ 노이즈가 매우 큰 경우 — SVD 기반 저랭크 근사로 어느 정도 완화 가능
